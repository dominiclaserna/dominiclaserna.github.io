NBA Player Clustering using K-Means

Tech: Python, Data Analysis, K-Means Clustering

This program classifies NBA players based on their game stats using the K-Means clustering algorithm. The main statistics used are Points per Game (PTS), Assists per Game (AST), and Rebounds per Game (REB). Users can input the number of clusters (k) for the classification.

Input: Dataset players_stat.csv for stats and players_id.csv for player names. Output: Generates output.txt containing initial/final centroids, number of iterations, and labeled dataset.
